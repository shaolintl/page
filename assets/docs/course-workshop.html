Each participant arrives with their daily work. They leave with a working AI agent built around a real problem they identified themselves — using the same four-pillar method from the flagship course. No pre-prepared scenarios. No demos. Real tools, built and shown to each other before the session ends.
Prerequisite: Completion of the AI Agents for Regulated Teams flagship course (10h), or equivalent hands-on AI experience.
How the session works
1
Surface the problems (30 min). Each participant describes two or three everyday tasks they do that involve data — documents, reports, alerts, or records. No preparation needed. The trainer helps identify which problem is the best candidate to build on.
2
Build (2h). Each participant builds an AI agent for their chosen problem using the four-pillar method: literacy (understanding the task), transparency (cited outputs), human oversight (checkpoint before anything is acted on), traceability (full log). The trainer circulates and guides.
3
Demo to each other (30 min). Each participant demonstrates the tool they built to the group, showing the input, the agent output with citations, and the human checkpoint. Peers ask questions. The trainer surfaces compliance implications.
What you leave with
  • A working AI agent built around a real problem from your own daily work
  • Outputs that are traceable and auditable: every conclusion cites its source
  • A human checkpoint wired in — no output is acted on before a person has reviewed it
  • The ability to build the next tool independently, without trainer support
  • A session record suitable for internal audit or regulatory inspection
